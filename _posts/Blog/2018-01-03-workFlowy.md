## 电脑上基本快捷键:
* Enter 换行。
* Tab 右缩进。
* Shift + Tab 左缩进。
* Shift + Enter 输入备注。
* Cmd + > 进入条目
* Cmd + < 退出条目
* Cmd + Enter 完成（ cmd + O  显示或隐藏完成项目）。
* Cmd + z 撤销。
* Cmd + Shift + z 不要撤销。
* Esc 搜索 (搜索后用方向键向下选择)。
* Cmd + / 查看所有快捷键。
* (Cmd 在你电脑上可能是 Ctrl 键,请试试)
